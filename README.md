## 💡 The Idea

Most people don’t have a money problem…
They have a **visibility problem**.

**Kharchameter** solves this by turning chaos into clarity —
giving users a simple, visual way to understand where their money actually goes.

---

## ✨ Features

* 💰 Add, delete & manage expenses in real-time
* 📊 Category-wise tracking (Food, Travel, etc.)
* ⚡ Instant UI updates using JavaScript
* 🎯 Minimal, distraction-free interface
* 📱 Responsive design for all screens

---

## 🧠 System Design (How it works)

```id="v7r7v7"
User Input → DOM Manipulation → Data Update → UI Re-render
```

* Uses **event-driven architecture**
* Efficient **state updates via JavaScript**
* Clean separation of structure, style, and logic
